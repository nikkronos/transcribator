# Roadmap: Transcribator

## Цель проекта

Локальная утилита для транскрибации аудио и видео в текст. Результат сохраняется в txt и json (с таймкодами) для последующего анализа через AI-агента (рефлексия по созвонам, подсветки, аналитика).

---

## Задачи в планах

### MVP (текущий фокус)
- [x] Локальное распознавание речи (русский): аудио и видео → текст — 2026-03-07
- [x] Вход: локальные файлы, поддержка нескольких файлов за один запуск — 2026-03-07
- [x] Выход: .txt (полный текст), .json (сегменты с таймкодами start/end/text) — 2026-03-07
- [x] Простой CLI без лишнего UI — 2026-03-07
- [x] Документация и подготовка к выносу в отдельный репозиторий — 2026-03-07

### После MVP (идеи)
- [ ] Опция выбора модели (размер/скорость/качество)
- [ ] Английский язык (опция)
- [ ] Улучшение UX (например, простой веб или GUI) — по решению позже
